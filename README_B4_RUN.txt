Important reminders for markers, to run the program appropriately:

1) Unzip everything to a folder (keep all .txt files in the same folder).

2) For GUI Version:  # Default file for marking
    - Either run "MainBS_GUI_For_Marking.py" directly,
    - OR run in VS Code  # Recommended

   For CLI Version: 
   - Either run "MainBookingSystem.py" directly,
    - OR run in VS Code  # Recommended
    
3) If an "Environment Incompatible" error message appears:  # If you have chosen GUI Version
   - Install Python >= 3.10 from python.org (includes Tk 8.6+), then retry.
   - If the problem still persists, please use the "MainBookingSystem.py" for marking (Some features are not supported in CLI mode)

Thank you for your corporation! Enjoy the program!
